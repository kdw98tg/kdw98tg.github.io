1. Git HTTP 버퍼 크기 증가

HTTP 연결이 끊어지는 문제를 해결하려면 Git의 HTTP 버퍼 크기를 증가시킬 수 있습니다. 아래 명령어를 실행하여 버퍼 크기를 설정해 보세요:

bash

Copy

git config http.postBuffer 524288000

이 명령어는 Git의 HTTP POST 버퍼 크기를 500MB로 설정합니다.
