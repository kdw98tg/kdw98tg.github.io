## QR pop() 시 날짜 데이터 제거

길이문제를 고려하여 앞에오는 날짜 데이터는 제거함

```python
    def pop_qr(self):
        """
        내부 큐에서 가장 먼저 들어온 QR 코드를 꺼내 반환합니다.
        큐가 비어있는 경우, 현재 타임스탬프를 조합하여 고유한 대체 NG 코드를 자동 생성합니다.
        """
        try:
            # qr은 현재, 날짜시간:QR코드 형태로 들어오므로, ":" 기준으로 분할하여 QR 코드 부분만 추출
            # 현재 QR양식과 다른 경우는 가정하지 않았음
            qr = self.qr_queue.get_nowait()
            qr_data = qr.split(":")[1]
            print(f"[QR] 반환(Pop): {qr_data}")
            return qr_data
        except queue.Empty:
            ng_code = "NG_" + datetime.now().strftime("%Y%m%d_%H%M%S_%f")
            print(f"[QR] 큐가 비어있음 (대체 코드 생성): {ng_code}")
            return ng_code
```
