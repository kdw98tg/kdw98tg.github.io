## 1. Node.js 최신 버전 설치하기

구글에 node js 를 치면, node js를 설치할 수 있는 페이지가 나옵니다. 거기서 각 프로젝트에 맞는 버전을 설치합니다.

## 2. React Project 생성하기

vscode 의 하단 터미널에 다음과 같은 명령어를 입렵합니다.
(대괄호는 기입하면 안됩니다. my-app-name에는 프로젝트 이름을 기입합니다.)

```bash
npx create-react-app [my-app-name]
```

이렇게 입력을 하게 되면, 터미널에서 프로젝트에 필요한 파일들을 다운받게 됩니다. 과정이 끝나면, 루트 폴더 안에 제가 설정한 프로젝트의 이름으로 폴더가 생기는것을 볼 수 있습니다.

![](/images/Pasted%20image%2020260105104920.png)


## 3. 프로젝트 실행.

새로 만든 폴더에 경로를 위치한 후에, vscode 하단 터미널에 다음과 같은 명령어를 입력합니다.

```bash
cd [my-app-name]
npm start
```

명령어가 실행되면, `localhost:3000`에 프로젝트가 열리게 됩니다.
